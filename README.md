一个基于 **Sun Valley ttk 主题** 的 Windows 11 风格批量图片重命名工具。

支持**拖拽导入**、**前缀添加**、**自动编号**、**重命名预览**、**撤销操作**，以及**保留原文件名 / 完全重新命名**两种模式。

---

## ✨ 功能特点

- ✅ 两种重命名模式：保留原文件名 or 完全重新命名
- ✅ 独立开关：前缀、自动编号
- ✅ 拖拽支持：直接拖入图片文件或文件夹
- ✅ 实时预览：修改配置即刻看到结果
- ✅ 覆盖保存 / 撤销操作
- ✅ 深色 / 浅色主题切换
- ✅ 单文件绿色打包，无需安装 Python 环境
